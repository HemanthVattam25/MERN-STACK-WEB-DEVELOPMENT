Getting started with HTML

This page is all about the learnings, implementations and practice for the first class of HTML. This page contains the following implementations

  1. h1 for Heading Tag
  2. p for paragraph
  3. ol for ordered list.

    4. Type of ordered list in numbers by default. if we want to change it's type. we can do that by mentioning in the attribute type = " "

    • i for lowercase romans and I for Uppercase Romans
    • a for lowercase letters and A for Uppercase letters
  4. ul for unordered list
  5. img for image
  6. a for anchor tag for links
  7. br for break
  8. inputfor input

    • email
    • file
    • hidden
    • image
    • month
    • number
    • password
    • radio
    • range
    • reset
    • submit
    • tel
    • time
    • url
    • week
  9. table for table
  10. thead for table head
  11. tr for table rows
  12. th for table headers
  13. tbody for table body
  14. td for table columns
