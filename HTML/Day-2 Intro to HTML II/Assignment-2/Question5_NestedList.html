Q5:Nested List Structure

  1. Learn continuously
    • Pay attention to Class

      • Listen to class carefully. Participate in discussions. Be Open to Questions. Write running notes

    • Know new things

      • Listen patiently. Be open to learn new concepts. practice after the class and clarify the doubts and get clear.

    • Code Everyday

      • Regularly code on the concepts covered. Be a professional at solving problems.

  2. Practice Daily
    • Solve assignments

      • solve the questions and the challenges given in the assignments

    • Write efficient Code

      • write clean code with formatting and no unnecessary things

    • Revise previous concepts

      • Implement old concepts in day to day practice.

  3. Solve New challenges
    • Face Challenges

      • Get ready with the ambiguous questions, challenges, confusing concepts, Research, Find the solution

    • Understand the problems

      • Research about the problem. Use tools like chatgpt, post queries in stackoverflow, Find similar questions over the internet.

    • Read Documentation

      • Research the documents about the questions, challenges and get clear with the questions for the clear foundation

  4. Maintain Good Portfolio
    • Push code regularly in gitHub

      • Regularly update and push code into gitHub. Be active and make contributions on daily basis.

    • Create Personal Projects

      • Make your own projects. Solve the problems innovatively and create a sustainable clean and attractive projects

    • Maintain personal website

      • Create a personal website with the necessary personal information, social profiles, github links, projects etc..
