This paragraph is used to check internal styling and external styling effect after inline styling.

The priority of styling for selectors is as given:

  • Inline Styling > (Internal Styling ~ External Styling)

    • NOTE: The styling priority between internal and external CSS happens as which file is saved before run. If we save HTML file and run, internal styling happens, If we save css file and run, External CSS styling happens

  • Inline styling > id (#) > class(.) > element (Tag name) > universal (*)
